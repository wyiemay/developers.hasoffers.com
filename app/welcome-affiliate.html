HasOffers Affiliate API V3 Documentation

The HasOffers Affiliate API (V3) exposes many functions of the HasOffers platform to end users. By interacting directly with our API, Affiliates can create and embed powerful widgets within their own applications.

To get started, make sure that you have requested and been approved for an Affiliate API V3 Key. To make a request to the API, simply provide your key as well as the Network ID in which your account is registered. Click on any function at the left to view the documentation and use the embedded query builder to execute requests directly to the HasOffers API.

Typical API request

http://api.hasoffers.com/v3/Affiliate_Affiliate.json?Method=getAccountManager&api_key=MY_API_KEY&NetworkId=SAMPLE
